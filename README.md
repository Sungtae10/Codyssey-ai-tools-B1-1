# Codyssey AI Tools Mission 1

## ChatGPT에게 일을 제대로 시키는 법 배우기

## 1. 프로젝트 개요

본 프로젝트는 **기술경영/CFM 논문 요약 및 연구 활용성 분석 AI 프롬프트 시스템**을 설계한 결과물이다.

선정한 업무 과업은 사용자가 기술경영, Corporate Foresight, Corporate Foresight Maturity(CFM), 혁신전략 관련 논문 초록 또는 본문 일부를 입력하면 AI가 다음 내용을 구조화하여 출력하도록 하는 것이다.

1. 연구 질문(RQ)
2. 방법론
3. 핵심 발견 3가지
4. 기술경영/CFM 연구와의 관련성
5. 한계 및 후속 연구 여지

본 프로젝트의 핵심 목적은 단순한 논문 요약이 아니라, 논문 내용을 연구계획서와 선행연구 정리에 활용 가능한 형태로 변환하는 것이다. 특히 원문에 근거한 사실과 연구 활용을 위한 추론을 구분하기 위해 `[추론]` 태그를 사용하도록 설계하였다.

---

## 2. 과업 선정 이유

기술경영 대학원생이나 연구자는 논문을 읽을 때 단순 요약보다 연구 질문, 방법론, 변수, 성과지표, 한계, 후속 연구 가능성을 빠르게 구조화하는 것이 중요하다.

특히 CFM 연구에서는 기존 논문이 이론적 근거, 측정 방식, 변수 설계, 예측타당성 검증에 어떻게 연결되는지 파악해야 한다. 따라서 본 프로젝트에서는 논문 초록을 입력하면 AI가 원문 사실을 요약하고, CFM 연구와의 연결 가능성은 `[추론]`으로 분리하여 제안하도록 설계하였다.

## 5. 사용 모델

본 프로젝트에서는 동일한 논문 초록과 동일한 프롬프트를 사용하여 다음 3개 LLM을 비교하였다.

| 모델 | 사용 목적 |
|---|---|
| Gemini 3.1 Pro | 안전장치 준수, 누락 정보 처리, 형식 준수 확인 |
| Claude Opus 4.8 | 사실/추론 구분, CFM 연구 연결성 확인 |
| GPT-5.5 | 연구 활용성, CFM 연구 설계 연결성, 최종 모델 후보 검토 |

최종 선정 모델은 **GPT-5.5**이다. 선정 이유는 CFM 연구와의 연결성이 가장 구체적이고, 공시 텍스트 기반 CFM 측정, t+1~t+3 성과 예측, 예측타당성 검증 등 실제 연구 설계에 활용 가능한 제안을 가장 풍부하게 제공했기 때문이다.
