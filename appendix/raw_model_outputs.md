# Appendix. Raw Model Outputs

공개 제출을 위해 개인 이름은 “사용자”, “연구자”, “본 연구”로 익명화하였다.

## 1. Gemini 3.1 Pro
- 입력 데이터 분석을 먼저 수행했다.
- 누락 정보 확인 질문 3개를 제시했다.
- 표본 크기, 대상 산업군, 상세 분석기법, 구체적 결과 수치가 초록에 없음을 확인했다.
- 원문에 없는 정보는 “원문에 명시 없음”으로 처리했다.
- CFM 관련성은 [추론] 태그로 분리했다.

## 2. Claude Opus 4.8
- 초록에 명시된 사실과 원문에 없는 정보를 표로 구분했다.
- 연구 설계, 측정 시점, FP 측정 모델을 명확히 정리했다.
- CFM Index 연구와의 연결성을 구체적으로 제안했다.
- 서베이 기반 측정과 공시 텍스트 기반 LLM 측정의 차별화 가능성을 [추론]으로 제시했다.
- 다만 5개 항목 외 “다음 행동”을 추가하여 형식 준수에서는 일부 감점했다.

## 3. GPT-5.5
- 연구 질문, 방법론, 핵심 발견을 구조적으로 정리했다.
- 표본, 분석기법, 통계모형은 “원문 초록에 명시 없음”으로 처리했다.
- CFM 연구의 이론적 정당화, 측정 논리, 예측타당성 검증 설계와 연결했다.
- DART 사업보고서 기반 CFM 측정, t+1~t+3 성과 예측, ROA, 영업이익률, Tobin’s Q, 시가총액 성장률 등을 후속 연구 설계와 연결했다.
- 일부 추론성 제안에서 [추론] 태그가 완전히 일관되지는 않아 v2 프롬프트에서 보완이 필요했다.
