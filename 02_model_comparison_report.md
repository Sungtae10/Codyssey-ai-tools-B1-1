# 02. Model Comparison Report

## 1. 비교 목적
동일한 논문 초록과 동일한 프롬프트를 3개 LLM에 입력하여, 기술경영/CFM 논문 요약 및 연구 활용성 분석 과업에 가장 적합한 모델을 선정한다.

## 2. 비교 대상 모델
| 모델 | 사용 환경 | 입력 조건 |
|---|---|---|
| Gemini 3.1 Pro | Web | 동일 프롬프트 |
| Claude Opus 4.8 | Web | 동일 프롬프트 |
| GPT-5.5 | Web | 동일 프롬프트 |

## 3. 평가 축
| 평가 축 | 설명 |
|---|---|
| 형식 준수 | 지정한 5개 항목을 지켰는가 |
| 사실 정확성 | 초록에 있는 내용을 왜곡하지 않았는가 |
| 누락 정보 처리 | 표본 수, 분석기법 등 없는 정보를 “명시 없음”으로 처리했는가 |
| [추론] 태그 준수 | 원문 사실과 연구 활용 추론을 분리했는가 |
| 연구 활용성 | CFM 연구에 적용 가능한 시사점을 제공했는가 |
| 한국어 학술 문체 | 대학원 연구 정리에 적합한 문체인가 |

## 4. 점수표

## 5. 모델별 근거 요약
### Gemini 3.1 Pro
누락 정보 처리와 안전장치 준수가 우수했다. 표본 수, 대상 산업군, 분석기법, 구체적 결과 수치가 초록에 없다는 점을 먼저 확인했다. 다만 CFM 연구와의 연결성은 비교적 일반적이었다.

### Claude Opus 4.8
사실과 추론의 구분이 명확했고, CFM Index 연구와의 연결성도 구체적이었다. 다만 지정된 5개 출력 항목 외에 추가 행동 제안을 생성하여 형식 준수에서 일부 감점했다.

### GPT-5.5
CFM 연구와의 연결성이 가장 구체적이었다. 공시 텍스트 기반 CFM 측정, t+1~t+3 성과 예측, 예측타당성 검증 등 후속 연구 설계 활용성이 높았다. 다만 일부 추론성 제안에서 [추론] 태그가 완전히 일관되지는 않았다.

## 6. 최종 선정 모델
**GPT-5.5**

## 7. 선정 이유
본 과업은 단순 요약이 아니라 연구 활용 가능한 논문 리뷰가 핵심이다. GPT-5.5는 CFM 연구 설계와 가장 직접적으로 연결되는 답변을 제공했다. 특히 공시 텍스트 기반 CFM 측정, t+1~t+3 성과 예측, ROA/영업이익률/Tobin’s Q/시가총액 성장률 등 연구 설계에 활용 가능한 제안을 제공했다. 일부 추론 태그 일관성 문제는 v2 프롬프트에서 보완 가능하다고 판단했다.
